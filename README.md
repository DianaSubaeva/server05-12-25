# Инструкция по использованию системы парсинга Dental-First.ru
# Быстрый старт
1. Установите Python 3.7+ (если еще не установлен).
2. Скачайте все файлы проекта в одну папку.
3. Откройте терминал/командную строку в этой папке.
4. Запустите программу:
   ```bash
   python main.py
   ```
# Автоматическая настройка
При первом запуске система автоматически:
- проверит установленные библиотеки;
- установит недостающие зависимости;
- запустит главное меню.
# Основные шаги работы
# Шаг 1: Полный запуск (рекомендуется)
В меню выберите "1. ЗАПУСТИТЬ ВСЕ (серверы + тест)"
- запустятся оба сервера;
- выполнится тестовый парсинг 2-х страниц;
- вы увидите сравнение результатов.
# Шаг 2: Проверка результатов
В меню выберите "4. Показать результаты"
- увидите количество найденных товаров;
- узнаете общую стоимость;
- посмотрите примеры товаров.
# Шаг 3: Экспорт данных
В меню выберите "5. Экспорт в CSV"
- Данные сохранятся в файлы:
  - `async_results.csv` - результаты асинхронного парсинга;
  - `threaded_results.csv` - результаты многопоточного парсинга.
# Шаг 4: Завершение работы
В меню выберите "6. Остановить все и выйти"
Или нажмите **Ctrl+C** в любой момент.
# Дополнительные возможности
# Запуск только серверов:
Выберите "2. Только запустить серверы" если хотите:
- протестировать серверы вручную;
- использовать свои скрипты для запросов;
- серверы будут доступны по адресам:
  - http://localhost:8080 (асинхронный);
  - http://localhost:8081 (многопоточный).
# Запуск теста вручную:
Выберите "3. Выполнить тест" если:
- серверы уже запущены;
- хотите протестировать разное количество страниц;
- система запросит количество страниц для парсинга.
# Важные заметки
# Если порты заняты:
Система автоматически найдет свободные порты (8082, 8083 и т.д.).
# Для остановки в экстренном случае:
1. Нажмите **Ctrl+C**.
2. Или закройте терминал/командную строку.
# Файлы результатов:
- `async_results.json` - результаты асинхронного парсинга;
- `threaded_results.json` - результаты многопоточного парсинга;
- `comparison_report.json` - сравнение производительности;
- `*_results.csv` - экспортированные данные в CSV.
# Тестирование вручную
Если серверы запущены, можете протестировать их напрямую:
# Асинхронный сервер (мгновенный ответ):
```bash
curl -X POST http://localhost:8080/parse \
  -H "Content-Type: application/json" \
  -d '{"url":"https://dental-first.ru/catalog","start_page":1,"end_page":2}'
```
# Многопоточный сервер (фоновый парсинг):
```bash
curl -X POST http://localhost:8081/parse \
  -H "Content-Type: application/json" \
  -d '{"url":"https://dental-first.ru/catalog","start_page":1,"end_page":2,"threads":3}'
```

Результаты многопоточного парсинга появятся в файле `threaded_results.json` через 10-20 секунд.
